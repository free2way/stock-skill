# Dashboard Export

Use this reference when turning several report JSON files into a simple watchlist view.

## Goal

Produce lightweight artifacts that are easy to review outside Codex:

- CSV for spreadsheets, sorting, and ad hoc analysis
- HTML for quick visual scanning

## Included Fields

The exporter keeps the view intentionally small:

- ticker
- analysis date
- score summary
- ranking score
- missing metrics count
- business quality
- growth durability
- key risks
- valuation context
- backtest summary

## Interpretation

- Use CSV when you want to sort or append your own columns.
- Use HTML when you want a compact daily or weekly watchlist snapshot.
- Re-export after new earnings, financing events, or major price-action reviews.
