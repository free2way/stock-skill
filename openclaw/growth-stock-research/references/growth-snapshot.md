# Growth Snapshot

Use this reference when you want a very short watchlist summary instead of the full dashboard.

## Goal

Compress a set of reports into a quick review note that answers:

- Which names rank highest right now?
- Which names look risky or incomplete?
- Which names deserve the next deep dive?

## Input

The snapshot script uses:

- report JSON files
- optional tag JSON from `tag_watchlist.py`

## Output

A markdown note with:

- top-ranked names
- names flagged for review
- a compact table for all names

## Best use

- morning review
- weekly check-in
- pre-earnings watchlist prep
