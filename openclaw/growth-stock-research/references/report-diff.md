# Report Diff

Use this reference when you want to compare two saved versions of the same stock report.

## Goal

Show what changed between review cycles without rereading the entire JSON.

## What the diff emphasizes

- analysis date change
- score change
- metric deltas
- missing-field changes
- narrative field edits

## Best practice

- save the pre-refresh report under a dated filename
- refresh the live report
- diff old vs new
- update your thesis notes only after checking the diff

## Interpretation

- Positive metric movement does not automatically mean the stock improved.
- Use the diff to locate the changes quickly, then decide whether they matter.
